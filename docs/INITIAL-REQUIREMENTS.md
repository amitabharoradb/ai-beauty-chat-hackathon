# AI Beauty Chat requirements

## Functional Requirements
The goal of this project is to develop a beauty chatbot for a fictitious company Uphora that sells high-end beauty products. it should save both short-term and long-term memory. Please figure out what should be saved.

## Non-Functional requirements
- for storing agent memory use Databricks Lakebase Autoscaling
- create fake data for 10000 customers, 3 beauty categories and 5-15 products in each category.  See sephora.com for samples
- use Databricks Apps for the chatbot
- use langraph
- deploy the agen on databricks apps and not as a model serving endpoint
